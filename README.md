## Introduction
- Online viewer that allows you to display the contents of the CSV file containing the MFT data extracted from the MFTParser tool
---

## Features
* **Vue 3** (Options API) for reactive UI.
* **JSON-based** data management.
* **Web Worker** integration for background processing.
* **Responsive Design** using pure HTML/CSS.
